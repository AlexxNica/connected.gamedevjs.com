Connected Gamedev

Game Development around Connected Devices

Make games, write docs, promote technology

Welcome to the Connected Gamedev website where you can find all the resources about making HTML5 games on Connected Devices. Remember: life is boring, let's build videogames!

This website and GitHub repository were created after the Mozilla Leadership Summit in Singapore to host all kinds of game development projects from the Connected Devices (Internet of Things) space. Feel free to dive in and create your own experiments, write the documentation and share the knowledge.
